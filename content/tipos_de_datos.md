# Booleanos

## Definicion
Es un tipo de dato que solo puede tener los valores de verdadero o falso.

## ¿Como se declaran?
Declaremos variables de tipo booleano

```{code-cell} python
verdad = True
print(verdad)
```

```{code-cell} python
falso = False
print(falso)
```
